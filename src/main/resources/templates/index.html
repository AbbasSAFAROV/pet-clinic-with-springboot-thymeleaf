Veterinary Open Source Application

The purpose of this application is assignment from ozgur soft. I used this application Spring Boot , PostgreSQL , Spirng Security and Thymeleaf for template engin.

Application has Crud Operation register and login, First you registered then you redirected login page, Application has two role admin and standard user, Standard User can be veterinary or employee , Standard role can do owner add , owner edit , pet add , pet edit only, can't delete anything Admin role can do everything.
